## Brand Overview

### Brand Essence

**Viridian Nutrition Platform** combines expert clinical nutrition coaching with intelligent AI technology to deliver personalized, evidence-based nutrition guidance at scale.

### Brand Values

- **Evidence-Based:** No gimmicks, no potions, no drinks, no pills
- **Expert-Led:** Clinical nutrition expertise at the core
- **Personalized:** Radically individualized for each client
- **Empowering:** Education over prescription
- **Accessible:** Scaling expertise through technology

### Brand Personality

- **Professional** yet approachable
- **Intelligent** yet friendly
- **Scientific** yet warm
- **Confident** yet humble
- **Modern** yet trustworthy

### Target Audience

**Primary:**
- Busy professionals (30-50 years)
- Athletes (20-40 years)
- Seniors (60+ years)
- Fitness enthusiasts

**Characteristics:**
- Smart, well-informed, motivated
- Value evidence over trends
- Seeking sustainable results
- UK-based (primarily)

### Clear Space

**Minimum clear space = X (height of icon)**

```
     X
  ┌─────┐
X │LOGO │ X
  └─────┘
     X
```

**Never:**
- Place elements within clear space
- Crowd the logo with other content
- Overlay logo on busy backgrounds without sufficient contrast

### Minimum Size

**Digital:**
- Minimum width: 120px (full logo)
- Minimum width: 32px (icon only)

**Print:**
- Minimum width: 25mm (full logo)
- Minimum width: 8mm (icon only)

### Logo Don'ts

❌ **Never:**
- Change logo colors
- Distort or stretch logo
- Rotate logo
- Add effects (drop shadow, gradient, outline)
- Place on low-contrast backgrounds
- Recreate or modify logo elements
- Use outdated logo versions

✅ **Always:**
- Use provided logo files
- Maintain aspect ratio
- Ensure sufficient contrast
- Respect clear space
- Use appropriate variant for background

---

### Typography Best Practices

**Hierarchy:**
- Use size and weight to create clear hierarchy
- Limit to 3 levels of hierarchy per screen
- Headings always use Georgia
- Body text always uses Arial

**Readability:**
- Body text minimum: 16px
- Line length: 50-75 characters optimal
- Line height: 1.5 for body, 1.25 for headings
- Paragraph spacing: 1em

**Emphasis:**
- Use bold (700) for emphasis
- Avoid using italic (not available)
- Use color sparingly for emphasis
- Never use all caps for long text

## Visual Style

### Design Principles

1. **Clean & Minimal**
   - Generous white space
   - Uncluttered layouts
   - Focus on content

2. **Professional & Modern**
   - Contemporary aesthetic
   - Subtle animations
   - Polished details

3. **Evidence-Based**
   - Data visualization
   - Charts and graphs
   - Scientific credibility

4. **Warm & Approachable**
   - Rounded corners (8-12px)
   - Friendly illustrations
   - Conversational tone

### Layout Grid

**8px Base Unit**
- All spacing increments of 8: 8, 16, 24, 32, 48, 64, 80
- Creates visual rhythm
- Ensures consistency

**Responsive Breakpoints:**
- Mobile: 390px - 767px
- Tablet: 768px - 1023px
- Desktop: 1024px+
- Wide: 1440px+

**Column Grid:**
- Mobile: 4 columns
- Tablet: 8 columns
- Desktop: 12 columns

---

### Border Radius

**Scale:**
- Small: 4px (badges, tags)
- Medium: 8px (buttons, inputs)
- Large: 12px (cards)
- XL: 16px (modals, dialogs)
- Full: 9999px (pills, avatars)

**Usage:**
- Consistent radius per component type
- Larger elements = larger radius
- Interactive elements = 8-12px
- Pills/chips = full radius

---

### Shadows

**Elevation System:**

```
None: No shadow
- Flat elements

Small: 0 1px 2px rgba(148, 19, 77, 0.05)
- Subtle depth
- Hover states

Medium: 0 4px 6px rgba(148, 19, 77, 0.1)
- Cards
- Dropdowns

Large: 0 10px 15px rgba(148, 19, 77, 0.1)
- Modals
- Popovers

XL: 0 20px 25px rgba(148, 19, 77, 0.15)
- Dialog overlays
- Important elements
```

**Note:** Shadows use primary color tint for brand consistency

## Photography Guidelines

### Style

**Characteristics:**
- Natural lighting
- Authentic moments
- Diverse representation
- UK context when relevant
- Focus on health and nutrition

**Subjects:**
- Real people (not stock models)
- Healthy food preparation
- Active lifestyles
- Coaching moments
- Progress celebrations

### Technical Specs

**Digital:**
- Format: WebP (primary), JPEG (fallback)
- Resolution: 2x for retina displays
- Optimization: TinyPNG, Squoosh
- Max file size: 200KB per image

**Composition:**
- Rule of thirds
- Negative space
- Centered for hero images
- Avoid clutter

### Treatment

**Color:**
- Natural color tones
- Slight warmth acceptable
- No extreme filters
- Consistent color grading

**Effects:**
- Subtle vignette OK
- No heavy processing
- Maintain detail
- Natural contrast

## Voice & Tone

### Brand Voice

**Core Attributes:**
- **Expert:** Clinical knowledge, evidence-based
- **Supportive:** Encouraging, non-judgmental
- **Clear:** Simple language, no jargon
- **Motivating:** Action-oriented, empowering

### Tone by Context

**Onboarding:**
- Welcoming and reassuring
- "We're here to help you succeed"

**Progress Tracking:**
- Celebratory and encouraging
- "Great work this week!"

**Educational Content:**
- Informative and accessible
- "Here's why this matters..."

**Error Messages:**
- Helpful and solution-focused
- "Let's fix that together"

---

### Writing Guidelines

**Do:**
✓ Use active voice
✓ Be concise and clear
✓ Use "you" and "your"
✓ Provide actionable advice
✓ Explain the "why"

**Don't:**
✗ Use medical jargon
✗ Make absolute claims
✗ Sound preachy
✗ Use guilt or shame
✗ Recommend supplements/pills
