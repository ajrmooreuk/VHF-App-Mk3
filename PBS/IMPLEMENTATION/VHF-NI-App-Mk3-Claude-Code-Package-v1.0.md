### üí° Key Technical Details

**Technology Stack:**
- Frontend: Next.js 14, React, Shadcn UI, Tailwind CSS
- Backend: Supabase (PostgreSQL + Auth + Edge Functions)
- AI: Anthropic Claude Sonnet 4, Claude Agent SDK
- Design: Figma, PT Sans from Google Fonts
- DevOps: Vercel, GitHub Actions

**Database Strategy:**
- 5 tables: clients, conversations, messages, recipes, meal_plans
- Schema.org ontologies in JSONB columns
- GIN indexes for fast JSONB queries
- Row-Level Security (RLS) enabled

**AI Architecture:**
- 2 Product Agents: Nutrition Advisor, Meal Planner
- 2 Platform Modules: Context Engineering, Value Engineering
- Three-tier context (Tier 1: cached indefinitely, Tier 2: cached per session, Tier 3: real-time)
- Prompt caching = 50% cost savings
